# Ecolingo — Execution Summary

**Product:** AI course compiler that turns a teacher's materials into a personalized, visual, game-like learning path.
**First course:** ECON 13210 — Introduction to Macroeconomic Models (proving ground, not a permanent limitation).
**Core loop:** teacher materials → concept map → personalized lessons → adaptive practice → mastery data → better next lesson.

This document is the programme's living index: scope decisions, the decision log, the risk register pointer, the source-provenance map, and the mapping of the 29 required output artifacts to files and owners.

## 7. First shippable experience — the bar

> A struggling learner can understand a difficult model through a personalized explanation and a precise interactive visual, demonstrate transfer, and be reviewed again before forgetting.

The slice in `app/` walks that path end-to-end for the Solow steady state: adaptive lesson steps → Solow Lab manipulation → transfer question → misconception-specific feedback → mastery update → explainable scheduled review.
